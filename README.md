# Shopping Trends Analysis Dashboard

## Project Overview
This project focuses on analyzing shopping trends data using Python.
The analysis is performed in Jupyter Notebook, and insights are visualized
using an interactive Power BI dashboard.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI Desktop & Power BI Service
- GitHub

## Project Files
- shopping_trends.ipynb → Data analysis notebook
- shopping_trends.csv → Dataset
- shopping_trend_pi.pbix → Power BI dashboard

## Power BI Dashboard

The Power BI dashboard is included in this repository as a `.pbix` file.
It can be opened locally using **Power BI Desktop**.
